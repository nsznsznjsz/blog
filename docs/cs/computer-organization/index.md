# 计算机组成原理

## 数据的表示

### IEEE 754

尾数全0，阶码全1表示无穷大

| 名称     | 尾数符号位 | 尾数 | 阶码 | 阶码偏移值 | 隐藏位 |
| -------- | ---------- | ---- | ---- | ---------- | ------ |
| 短浮点数 | 1          | 7    | 23   | 127        | Y      |
| 长浮点数 | 1          |      |      |            | Y      |
| 临时实数 | 1          |      |      |            | N      |
